# 📊 Microsoft PL-300 Certification Journey
> **Goal:** Study repository, labs, and technical documentation for the Power BI Data Analyst exam.

This project follows an agile methodology managed via **Jira** (Kanban) and version control via **GitHub** to track my progress from intermediate to advanced levels.

---

## 🏗️ Repository Structure

The repository is organized according to the official exam domains:

* **[01-Data-Preparation](./Preparation/):** Data sources, Power Query, basic M Language, and Data Profiling.
* **[02-Modeling-DAX](./Modeling_DAX/):** Star Schema, Cardinality, Row-Level Security (RLS), and complex DAX measures.
* **[03-Visualization](./Report_visualization/):** Storytelling, Dashboard Design, AI Visuals, and Accessibility.
* **[04-Cloud-Service](./Service_Cloud/):** Workspaces, Gateways, Apps, Incremental Refresh, and Governance.
